## 用户

### 1. 用户组管理

- #### 创建用户组
  ```bash
  groupadd 用户组名
  ```

- #### 删除用户组
  ```bash
  groupdel 用户组名
  ```

### 2. 用户管理

- #### 创建用户
  ```bash
  useradd [-g -d] 用户名
  ```
  #### `-g` 指定用户的组，如果组不没有设定就会创建一个和用户名同名的组，如果指定的group不存在就会报错
  #### `-d` 指定用户法home路径，不指定，home目默认在/home/用户名
  #### `-p` 给一个新用户设置密码

- #### 删除用户
  ```bash
  userdel [-r] 用户名
  ```
  #### `-r` 删除用户的同时删除home目录，如果不加`-r`将不会删除home目录

- #### 查看用用户所属组
  ```bash
  id 用户名
  ```

- #### 修改用户所属组
  ```bash
  usermod -aG 用户组 用户名 将指定的用户组
  ```

### 3.查看组
