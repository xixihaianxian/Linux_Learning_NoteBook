  - ##### `apt`/`apt-get`用于自动化安装linux软件，并可以自动解决依赖问题

  - ##### 语法
    ```bash
    agt-get/agt [-y] install/remove/search 软件名陈
    ```

  - ##### **注意**
    | 操作 | `apt` 命令 | `apt-get` 命令 |
    |------|------------|----------------|
    | 更新软件包列表 | `apt update` | `apt-get update` |
    | 升级已安装包 | `apt upgrade` | `apt-get upgrade` |
    | 安装包 | `apt install pkg` | `apt-get install pkg` |
    | 删除包（保留配置） | `apt remove pkg` | `apt-get remove pkg` |
    | 完全删除包（含配置） | `apt purge pkg` | `apt-get purge pkg` |
    | 自动删除无用依赖 | `apt autoremove` | `apt-get autoremove` |
    | 搜索包 | `apt search keyword` | ❌（需用 `apt-cache search`） |
    | 查看包信息 | `apt show pkg` | ❌（需用 `apt-cache show`） |
    | 列出可升级包 | ✅（`apt list --upgradable`） | ❌（不直接支持） |
